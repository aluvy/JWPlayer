## SETUP

SETUP API는 플레이어를 생성하고 설정 정보를 제공하는데 사용합니다.

- [**ready**](./setup.md#onready)
  - 플레이어가 초기화되어 재생할 준비가 된 시점을 나타냅니다. 이것은 API 호출이 가장 먼저 이루어져야 하는 시점입니다.
- [**remove**](./setup.md#onremove)
  - 플레이어가 `jwplayer().remove()` 를 통해 삭제될 때 트리거됩니다.
- [**setupError**](./setup.md#onsetuperror)
  - 플레이어를 설정할 수 없을 때 발생합니다.
